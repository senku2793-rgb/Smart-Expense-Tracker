# Smart Expense Tracker (Web App)

A web-based application to track daily expenses, categorize spending,
calculate monthly totals, and visualize expense distribution.

## Features
- Add daily expenses
- Category-based tracking
- Automatic total & percentage calculation
- Interactive charts
- GitHub Pages compatible

## Technologies
- HTML
- CSS
- JavaScript
- Chart.js

## SDG Alignment
- SDG 8: Decent Work & Economic Growth
- SDG 12: Responsible Consumption

## Deployment
Hosted using GitHub Pages (static web application).
